Features

Native Code Completion WIP

intellij-rust features built from scratch code completion, leveraging IntelliJ Platform capabilities. Ctrl+Space invokes completion. A decent amount of completion intelligence is already implemented, but it's still still at early stages of development. We do not plan to use racer

Navigate Everywhere

  • Press Ctrl+B to jump to definition
  • Ctrl+N finds any struct or enum by name
  • Ctrl+Alt+Shift+N finds any symbol (functions, traits, modules, methods, fields) by name
  • Ctrl+U brings you to the parent of the current module

For the in-depth guide of available features, please refer to the documentation.
